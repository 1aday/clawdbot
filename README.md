# ClawdBot

Personal multi-channel AI assistant for WhatsApp, Telegram, Slack, Discord, iMessage with voice and canvas support.

## What it does

ClawdBot is a unified AI assistant that connects across multiple messaging platforms. It processes messages from WhatsApp, Telegram, Slack, Discord, and iMessage through a single gateway, providing intelligent responses powered by OpenAI. It supports voice messages and canvas-based visual interactions.

## Tech Stack

- TypeScript
- Multi-channel Gateway
- OpenAI

## Getting Started

```bash
git clone https://github.com/1aday/clawdbot.git
cd clawdbot
npm install
npm run dev
```

## Environment Variables

- `OPENAI_API_KEY` - OpenAI API key
- `WHATSAPP_TOKEN` - WhatsApp Business API token
- `TELEGRAM_BOT_TOKEN` - Telegram Bot API token
- `SLACK_BOT_TOKEN` - Slack Bot OAuth token
- `DISCORD_BOT_TOKEN` - Discord Bot token
